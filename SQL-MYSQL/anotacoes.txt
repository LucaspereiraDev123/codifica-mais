Para acessar o MySQL por linha de comando, começamos abrindo o Prompt de comando da nossa máquina.

Agora vamos acessar o subdiretório do MySQL.

Começamos digitando *cd* e clicando no enter;

Na sequência digite:

cd “Program Files” e digite enter;

cd MySQL digite enter;

cd “MySQL Server 8.0” digite enter;

cd bin digite enter;

dir mysql.exe digite enter;

Agora acessamos o executável do MySQL, então vamos fazer o login:

mysql -h localhost -u root -p digite enter;

Ele pedirá sua senha cadastrada durante a instalação da ferramenta, então você deve digitá-la, depois digite enter novamente. Agora já podemos criar os scripts SQL!

Começamos criando um banco de dados teste com o seguinte comando:

CREATE DATABASE Teste; depois digitamos enter;

Você pode conferir a criação do banco acessando a interface do MySQL Workbench e indo até a lateral esquerda e atualizar a aba de schemas.

Fizemos mais um comando para acessar os registros de uma das tabelas do banco de dados sakila da seguinte forma:

USE sakila digite enter;

SELECT * FROM actor; digite enter e você terá todos os registros desta tabela.

Para sair do MySQL digite exit
